== Description ==

This is a simple plugin with a few cusomization options that adds a back to top link at bottom of page.

== Installation ==

1. Upload `elite-back-to-top` to the `/wp-content/plugins/` directory or install via the 'Add New' plugin area in administration section.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Customize via the settings page.
4. Enjoy.
